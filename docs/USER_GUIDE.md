# JARVIS AI User Guide

Welcome to **J.A.R.V.I.S. AI** — your Real-Time Personal Desktop AI Assistant.

---

## Quick Start

1. Double click `run.bat` or run `.\run.ps1` in PowerShell.
2. If launching for the first time, JARVIS will prompt you for your **Gemini API Key**.
3. Once loaded, the arc-reactor interface will initialize.

---

## Voice & Text Control

You can speak directly to JARVIS using your microphone or type commands in the desktop prompt window:
- **"Open Chrome and search for quantum computing"**
- **"What is the system CPU and memory usage?"**
- **"Create a python script that cleans CSV files"**
- **"Take a screenshot and analyze the active window"**
- **"Remind me to take a break in 30 minutes"**

---

## Configuration & Logs

- **API Keys**: Stored safely in `config/api_keys.json`.
- **Logs**: Detailed execution logs and crash reports are written to `logs/crash.log` and `logs/app.log`.
- **Memory Store**: Personal identity details and saved user context reside in `memory/long_term.json`.
