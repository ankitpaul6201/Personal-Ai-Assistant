# Supported Operating Systems & Python Versions

## Operating System Matrix

| OS Platform | Versions Supported | Status |
| :--- | :--- | :--- |
| **Windows** | Windows 10, Windows 11 | Fully Supported (DirectShow / Win32 API) |
| **macOS** | macOS 12+ (Monterey, Ventura, Sonoma) | Fully Supported (AVFoundation / Apple Silicon) |
| **Linux** | Ubuntu 22.04+, Fedora 38+, Arch | Fully Supported (V4L2 / PulseAudio / ALSA) |

---

## Python Version Support

| Python Version | Release Status | Recommended |
| :--- | :--- | :--- |
| **Python 3.12** | Fully Supported | ⭐ Recommended |
| **Python 3.11** | Fully Supported | Supported |
