# 📥 J.A.R.V.I.S. AI - Installation & User Guide

Welcome to **J.A.R.V.I.S. AI**, your real-time desktop AI assistant.

---

## 💻 Option 1: Standalone Windows Executable (No Python Required)

1. Download the `release/` folder or `Jarvis_Setup.exe` installer.
2. If using `Jarvis_Setup.exe`:
   - Run the installer.
   - Follow the setup wizard to install to Program Files.
   - Launch J.A.R.V.I.S. from Desktop shortcut or Start Menu.
3. If using the portable folder (`release/Jarvis/`):
   - Double-click `Jarvis.exe` to launch.

---

## 🐍 Option 2: Running from Python Source Code

1. Install **Python 3.11** or **3.12**.
2. Clone or extract the project directory.
3. Run setup wizard:
   ```bash
   python setup.py
   ```
4. Start J.A.R.V.I.S. AI:
   ```bash
   python main.py
   ```

---

## 🔑 Initial Configuration

When J.A.R.V.I.S. starts for the first time:
1. Enter your **Gemini API Key** (from [Google AI Studio](https://aistudio.google.com/)).
2. Configure Assistant Name and User Name in the Settings overlay (`F4` or gear icon).
